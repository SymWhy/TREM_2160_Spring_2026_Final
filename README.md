# TREM_2160_Spring_2026_Final
"Get Context" is a Firefox extension that searches for a highlighted string of words on various reputable fact checking websites. The intention of this app is to give the user context and information that might be otherwise absent from an online news article.

This program works by removing punctuation from a highlighted phrase, and opening a link with the phrase directed to the user's chosen fact checking search engine.

This quick search method is meant to exist alongside the "doomscroll" by allowing the user to search without going to another website first. It is meant to make information and context more accessible to users with shorter attention spans.
